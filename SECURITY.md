## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in OTU, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to [contact@opentutorials.org](mailto:contact@opentutorials.org).

You should receive a response within 24 hours. If for some reason you do not, please follow up via email to ensure we received your original message.
